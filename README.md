# MedInventário

Projeto desenvolvido no âmbito da unidade curricular de Sistemas de Informação e Base de Dados Aplicados à Saúde.


## Estrutura do projeto

- public/: páginas públicas acessíveis sem autenticação
- private/: páginas da área reservada
- assets/css/: ficheiro de estilos 
- assets/js/: ficheiro JavaScript 
- assets/img/: imagens utilizadas
- assets/bootstrap/: ficheiros Bootstrap
- assets/fontawesome/: ícones Font Awesome

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome
